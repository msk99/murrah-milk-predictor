About the Model

This tool predicts the Total Lactation Milk Yield (TLMY) of Murrah buffaloes using an XGBoost (Extreme Gradient Boosting) machine learning model. The model was trained on data from 182 Murrah buffalo lactation records collected at ICAR-CIRB (Central Institute for Research on Buffaloes), Hisar, India.

Three easily measurable predictors are used: Peak Yield (PY) in kg/day, Lactation Length (LL) in days, and Dry Period in days. Together, these capture the shape and duration of the lactation curve and the inter-calving rest period, enabling accurate estimation of total milk output for a given lactation.

The prediction includes a 95% bootstrap prediction interval derived from 50 bootstrap resamples per cross-validation fold, giving a measure of the uncertainty around each estimate.

The model was trained on 182 Murrah buffalo lactation records from the IASRI dataset (file: Julyiasri.xls). All variables are complete with no missing values.

Cross-Validation & Confidence Intervals

The optimal number of boosting rounds (76) was determined by 10-fold cross-validation with early stopping (patience = 50 rounds). The model was then evaluated using a separate 10-fold CV where, for each fold, 50 bootstrap resamples of the training set were drawn, refitted, and used to predict the held-out test set. The 2.5th and 97.5th percentiles of the bootstrap predictions form the 95% prediction interval.

For this web deployment, the prediction interval is approximated using a second-degree polynomial fitted to the CV bootstrap results: CI bound = a0 · x2 + a1 · x + a2, where x is the predicted TLMY value.
